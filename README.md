# CCMA Prep

A static GitHub Pages/PWA study app for CCMA exam practice.

## Production files

This clean release contains only the files needed to run the app:

- `index.html`
- `style.css`
- `app.js`
- `questions.json`
- `service-worker.js`
- `manifest.webmanifest`
- favicon and PWA icon assets

Release: `1.0.0`
